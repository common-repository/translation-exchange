== Description ==
Translate your WordPress into any number of languages using our integrated solution.

* Works with the most popular localization plugins, like WPML, Polylang and qTranslate X
* Provides its own JavaScript solution for website translations
* Supports over 600 languages and dialects
* Offers a Universal Translation Memory (UTM) with millions of translations
* Provides in-context translation tools
* Contains crowd-sourced translation option
* Connects you to a translator marketplace of over 50,000 professional translators
* Automatically extract new content and pull in latest translations
* Monitors translation progress and provides visitor analytics

== Installation ==

The installation instructions are based on the plugin of your choice.

If you decided to use our JavaScript solution, follow these steps:

1.  [Sign up for a free Translation Exchange account](https://gateway.translationexchange.com/signup)
1.  Create a new project and copy your project key.
1.  Install this plugin and open the settings page from the admin menu.
1.  Paste your project key and save the settings.
1.  Add Language Selector widget from "Appearance > Widgets" section.
1.  Visit your site and see it in other languages.

= Minimum Requirements =

* WordPress 3.0 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
